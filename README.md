# Task-3-PrinceRamteke
DecodeLabs Java Project 3 - ATM Interface

### Project Description

This project is a console-based ATM Interface developed using Java and Object-Oriented Programming (OOP) concepts.

The application simulates basic ATM operations such as checking account balance, depositing money, withdrawing money, and exiting the system. It demonstrates the use of classes, objects, encapsulation, methods, conditional statements, loops, and user input handling.
